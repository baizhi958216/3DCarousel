# 3D 旋转木马 轮播图

用法:

```html
<div id="carousel">
  <div class="carousel-item">
  </div>
  <div class="carousel-item">
  </div>
  <div class="carousel-item">
  </div>
</div>
```

```js
initCarousel({
  xRadius: 350,
  yRadius: 35,
  farScale: 0.7,
  speed: 4,
  itemClass: "carousel-item",
  handle: "#carousel",
});
```
